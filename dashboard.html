Announcement

The first thing you see on the gateway after logging in is the Announcement Image Slider.

									Tables Affected:
									posts
								
									Models Used:
									root/app/Posts.php
								
									Query (Eloquent):
									
Posts::where('enabled', '=', '1')->get()

New Hires

The first column in the last row is the New Hires column, it will show the newly added Employees from the Gateway, from the latest to the oldest.

									Tables Affected:
									employee_info
								
									Models Used:
									root/app/User.php
								
									Query (Eloquent):
									
User::allExceptSuperAdmin()->orderBy('prod_date', 'DESC')->paginate(5)

								
									Functions Used:
									
HomeController::newhires
Comments
  • Using the paginate function, we return the 5 latest Employees, and show it on the list.
  • After a certain button is click, the function HomeController::newhires will get the another 5 Employees who are added to the Gateway.

Birthday Celebrants

The second column in the last row is the Birthday Celebrants column, it will show the birthday celebrants for that specific month.

									Tables Affected:
									employee_info
								
									Models Used:
									root/app/User.php
								
									Query (Eloquent):
									
User::whereRaw('MONTH(birth_date) = '.date('n'))->whereRaw('deleted_at is null')->where('status','=',1)->orderByRaw('DAYOFMONTH(birth_date) ASC')->get()
Comments
  • Get all Active Employees from the database, and order it by the dates

eLinkgagement Activities

The last column in the last row is the eLinkgagement Activities column, it will show all the upcoming activities for the next 30 Days.

									Tables Affected:
									elink_activities
								
									Models Used:
									root/app/ElinkActivities.php
								
									Query (Eloquent):
									
ElinkActivities::getActivities()
Comments
  • Get the activities for the next 30 days, and order them by the Activity Date
